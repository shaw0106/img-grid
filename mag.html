Magazine Spread

I wanted to create a recipe magazine that I would want to read. I always find it frustrating when I have to write down the recipe from a magazine. I have tackled this design problem by placing all the info you need on a card that can be easily removed all together and kept seperate. Each card has a recipe on both sides so the maximum amount of information can be retained.
